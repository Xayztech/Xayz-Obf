TUTORIAL LOGIN & AKSES

METODE 1: LOGIN OTOMATIS (WIDGET)

1. Klik tombol "Log in as..." berwarna biru di layar utama.

2. Telegram akan mengirim pesan konfirmasi di aplikasi Telegram HP/PC kamu.

3. Buka chat dari "Telegram" (Service Notification).

4. Klik tombol "Confirm" atau "Accept" di chat tersebut.

5. Website akan otomatis reload dan kamu berhasil masuk.

METODE 2: LOGIN DENGAN NOMOR HP

1. Jika tombol biru tidak muncul, akan ada kolom input Nomor HP.

2. Masukkan nomor HP yang terdaftar di Telegram (awalan +62...).

3. Telegram akan mengirim pesan permintaan akses.

4. Konfirmasi di aplikasi Telegram.

METODE 3: GUEST MODE (TANPA LOGIN)

1. Klik tombol "GUEST MODE" berwarna abu-abu.

2. Kamu bisa langsung menggunakan fitur Encrypt.

3. Peringatan: History file tidak akan disimpan jika kamu refresh halaman.
